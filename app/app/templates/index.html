Syntax: the values of the variables are entered in text area. The elements inside the IF block are optional and can be omitted. Examples:

PORT;SPEED;GROUP;DESCRIPTION;VLAN => our arguments may be:
gi1/1;
10M;
3;
Link to provider;
10,20-30

If channel-group number is omitted => our arguments may be:
gi1/1;
10M;
;
Link to provider;
10,20-30

Or it is possible write all variables in one string:
gi1/1;10M;3;Link to provider;10,20-30

To generate same template with different arguments use # as separator:

gi1/1;10M;3;Link to provider;10,20-30#
gi2/1;20M;;LAN;1,8,19

Result:

!default interface gi1/1
interface gi1/1
 description Link to provider
 switchport trunk allowed vlan 10,20-30
 switchport mode trunk
 channel-group 3 mode active
 service-policy output 10M
 service-policy input 10M
